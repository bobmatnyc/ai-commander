# macOS Desktop Application with Guided Setup Experience

## Description
## Vision

Create a distributable macOS application that provides a simple GUI for Commander with guided setup for new users.

Users download a `.dmg`, drag to Applications, launch, and Commander guides them through:
1. Entering their OpenRouter API key
2. Installing/configuring AI coding adapters (Claude Code, Aider, etc.)
3. Setting up communication channels (Telegram, etc.)
4. Managing running sessions

## User Journey

```
Download Commander.app
        ↓
First Launch → Setup Wizard
        ↓
┌─────────────────────────────────┐
│  Welcome to Commander!          │
│                                 │
│  Enter your OpenRouter API key: │
│  [________________________]     │
│                                 │
│  [Continue]                     │
└─────────────────────────────────┘
        ↓
┌─────────────────────────────────┐
│  Install AI Adapters            │
│                                 │
│  ☑ Claude Code (recommended)    │
│  ☐ Aider                        │
│  ☐ Custom...                    │
│                                 │
│  [Install Selected]             │
└─────────────────────────────────┘
        ↓
┌─────────────────────────────────┐
│  Communication Channels         │
│                                 │
│  ☐ Telegram Bot                 │
│    [Setup Guide]                │
│                                 │
│  [Skip] [Configure]             │
└─────────────────────────────────┘
        ↓
Main Dashboard (session management)
```

## Technical Approach

### Framework Options

- **Tauri** (Rust + WebView) - Native performance, small binary, web UI
- **egui** (Pure Rust) - No web dependencies, immediate mode GUI
- **Swift + Rust FFI** - Native macOS look and feel

### Architecture

```
commander-gui/
├── src-tauri/          # Rust backend (uses commander-core)
├── src/                # UI (Svelte/React or native)
└── resources/
    └── installer/      # Adapter installation scripts
```

## Sub-Tasks

### 1. Installer Framework
- Adapter detection (is Claude Code installed?)
- Adapter installation (npm install -g @anthropic-ai/claude-code)
- Dependency management (Node.js, Python, etc.)
- Secret storage (Keychain integration)

### 2. Setup Wizard
- OpenRouter API key entry and validation
- Adapter selection and installation
- Telegram bot setup flow
- Initial project configuration

### 3. Main Dashboard
- Session list with status
- Quick connect/disconnect
- Message preview
- Notification integration

### 4. Distribution
- Code signing for macOS
- Notarization for Gatekeeper
- Auto-update mechanism
- DMG installer creation

## Dependencies

- commander-core crate (ISS-0013)
- Framework installer system
